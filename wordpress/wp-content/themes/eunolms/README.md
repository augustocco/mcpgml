# EUNO LMS Theme

Tema personalizado para WordPress optimizado para plataformas de aprendizaje (LMS).

## Características

- Diseño moderno y responsivo
- Compatibilidad con widgets personalizados
- Soporte para thumbnails de posts
- Menú de navegación principal
- Barra lateral y pie de página con widgets
- Optimizado para SEO
- Compatible con WooCommerce (si se instala el plugin)

## Instalación

1. Subir la carpeta `eunolms` a `/wp-content/themes/`
2. Ir a Apariencia > Temas en WordPress
3. Activar el tema "EUNO LMS"

## Estructura de Archivos

- `style.css` - Hoja de estilos principal
- `functions.php` - Funciones del tema
- `index.php` - Plantilla principal
- `header.php` - Encabezado del sitio
- `footer.php` - Pie de página
- `sidebar.php` - Barra lateral con widgets
- `screenshot.png` - Imagen de vista previa del tema

## Personalización

El tema soporta las siguientes opciones de WordPress:
- Logo personalizado
- Imagen de cabecera personalizada
- Color de fondo personalizado
- Menús personalizados
- Widgets en barra lateral y pie de página

## Compatibilidad

- WordPress 5.0+
- PHP 7.4+
- Compatibilidad con LearnDash LMS
- Compatibilidad con WooCommerce
